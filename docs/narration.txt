An agent that forgets between sessions is useless for recurring back-office work. Archon gives it persistent, queryable memory across sessions — live on Alibaba Cloud on real Qwen models.

First, proof it's real: the health endpoint shows Qwen text-embedding-v4 and qwen-plus, with genuine 1024-dimension vectors, and a live badge counts the memories persisted.

Every event, finding and insight is embedded with text-embedding-v4 and written to pgvector on Alibaba — durable, not in-process. Recall is hybrid: dense plus BM25 fused with reciprocal rank fusion, a cross-encoder rerank, then a grounded qwen-plus answer.

Now, live in the Explorer. In Session A an agent commits fused financial facts — invoices, cash positions, workforce cost. Then the session ends; nothing stays in process.

Session B is a fresh client with only a question: what did it truly cost to employ the team, versus what left the bank? Qwen embeds it, searches pgvector by cosine, and qwen-plus returns a grounded, cited answer — fifteen thousand eight hundred euros of true cost against ten thousand from the bank — written in one session, recalled by meaning in another.

Now the differentiator: a memory that audits itself. Two sessions wrote one record with two values. Mem0 and Zep would silently overwrite; Archon never mutates — it detects the contradiction and recommends which to trust, with a rule and rationale.

It even audits meaning, not just fields: notes that a vendor pays on time, then is chronically late, share no number to compare, so an authenticated semantic pass asks qwen-plus if they contradict. Four core memory operations are also exposed as typed MCP tools; Streamable HTTP is authenticated and tenant-scoped.

And it's measured: five of five contradictions detected, zero false positives, a recommender correct four of four; retrieval Recall at three rises from ninety to ninety-six point seven percent, at full answer correctness.

It runs live on Alibaba Cloud — an ECS instance in Singapore beside a self-hosted pgvector container. Real Qwen, measured, open source under MIT — not a wrapper.

And the same memory is queryable right in the browser — with grounded, cited answers.
